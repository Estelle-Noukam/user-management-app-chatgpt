# User Management App

## Description

Application web de gestion des utilisateurs développée avec Flask.

## Fonctionnalités

- Création de compte
- Connexion et déconnexion
- Gestion des rôles (Admin / Utilisateur)
- Profil utilisateur
- CRUD complet des utilisateurs
- Validation des entrées côté serveur

## Technologies

- Python
- Flask
- SQLite
- HTML
- CSS
- Docker

## Installation

pip install -r requirements.txt

python app.py

## Docker

docker-compose up --build

## Accès

http://localhost:5000
